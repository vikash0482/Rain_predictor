# 🌦️ Rain Predictor App

A full-stack web application that predicts whether it will rain tomorrow based on user input city.

## 🚀 Features
- Enter any city name
- Predict rain for tomorrow
- Clean UI with EJS templating
- API integration using Axios

## 🛠️ Tech Stack
- Node.js
- Express.js
- EJS
- Axios
- CSS

#Installation .....
* bash
* npm install
* nodemon index.js


## 🌐 API Used
This project uses the OpenWeatherMap API to fetch weather forecast data.

- API: https://openweathermap.org/api
